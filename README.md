## View Presentation

Install slides, via:

```sh
go install github.com/maaslalani/slides@c18781bb27646334057133af94ef6fe62ff2bb92
```

In a terminal, run:

```sh
slides SLIDES.md
```

Alternatively, the documentation is markdown, so you can read it with any
texteditor.
